Data Types
The simplest form of datatypes are non-parameterized. The constructors of a non-parameterized data-type can be given as a comma-separated list. 
        
          data Bool { True, False }
This is just a shorthand notation for 
        
          data Bool { True: Bool, False: Bool }
The shorthand notation which omits the type instantiation is only allowed for non-parameterized types. If the type has parameters, then the full form must be written out: 
        
          data Ty { True: Bool, False: Bool }
If we want to form the type of lists of some element type, we need to parameterize the type over elements of the type universe Type. 
        
          data List(a: Type) { Nil(a: Type): List(a), Cons(a: Type, x: a, xs: List(a)): List(a) }
